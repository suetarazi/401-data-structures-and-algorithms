## Tree Intersection
*Author: Sue Tarazi*

---

### Problem Domain

Write a method that takes in two binary trees and returns a set of values found in both trees.

---

### Inputs and Expected Outputs: 

| Input | Expected Output |
| :----------- | :----------- |
| two binary trees | a list of values |
 

---

### Big O

| Time | Space |
| :----------- | :----------- |
| O(n) | O(n) |

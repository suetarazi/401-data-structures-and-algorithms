## Left Join
*Author: Sue Tarazi*

---

### Problem Domain

Take in two hashmaps and perform a left join, returning the values that appear in both and Null for those that do not.

---

### Inputs and Expected Outputs: 

| Input | Expected Output |
| :----------- | :----------- |
| two hashmaps | one hashtable |
 

---

### Big O

| Time | Space |
| :----------- | :----------- |
| O(n) | O(n) + n |
